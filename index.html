  • Research into AI chess bots that could be used to complete the game prior to successfully completing the opening.
  • Tracking, analysis, and presentation of project metrics using Jira including burndown charts, task completion, hour logging, and feature documentation.
Project Framework & Language(s) This project was built using ReactJS NodeJS HTML CSS.
